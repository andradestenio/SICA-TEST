# SICA — v2.49

## Registro de presença

- Salvamento automático mantido.
- Botão Salvar registro abaixo da lista.
- Após salvar, os campos ficam bloqueados.
- Botão Editar registro libera os campos novamente.
- Mudança de disciplina, grupo, semana, data ou atividade reabre o modo de edição.

## Gestão de acessos

- Administradores e professores podem abrir a página.
- Professores têm acesso somente para consulta.
- Apenas administradores podem aprovar, reprovar, remover ou alterar perfis.
- Perfis disponíveis: Administrador, Professor e Usuário.
- Acesso direto por hash é bloqueado para outros perfis.
